Most Used Commands

New Branch

git checkout -b ticket-1234
git push -u origin ticket-1234

Pushing Updates

git add.
git commit -m "etc #1234" or just git commit -am "etc #1234"
git push origin ticket-1234 or just git push

Updating & Feeding Branches

git checkout develop
git pull
git checkout ticket-1234
git merge develop

Delete Branches

git branch -d {the_local_branch} (use -D instead to force deletion )
git push origin :{the_remote_branch} ex:(git push origin :ticket-1234)
git remote update origin --prune
git branch --merged master | grep -v '^[ *]*master$' | xargs git branch -d delete local branches faster

Common Commands

history shows history commands used
git clone {url}
git log -p --author=luisperez search commit by author
git diff {branch} -- "*.tpl" show only template file changes

Tags

git tag --list shows all tags
git show ### shows info of chosen tag

Can't Access Branch

git remote update
git fetch
git checkout --track origin/{BRANCH-NAME}

Cherry Pick

you will need to get you sha commits and apply them in order from oldest to newest
git checkout {the branch}
git cherry-pick c90fd66 complete sha per commit
to reverse cherry-pick
git revert --strategy resolve [SHA]

Check For Lang Changes

git diff {branch} --name-only displays file name changes
git diff alpha full/file_path/for/lang.php exports all changes into a file
This is a way to output changes by comparing branches
git diff {branch} > {name of file}.diff saves changes of current ticket
git diff hotfix-2016.12.07 > lang_changes/test.diff another example
git diff develop application/language/spanish/core_lang.php > spanish_core.diff checking differences per file
git diff develop --name-only *_lang.php shows only filtered file names
git diff develop..master > test.diff another way to compare two branches
To use my lang_key_comparison script
./dev_ops/lang_key_comparison.sh
Follow the prompts from the script and the changes will be outputed
